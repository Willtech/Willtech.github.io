## 🔧 Active Projects

- [**Conway's Game of Life**](https://github.com/Willtech/game-of-life)  
  A beautiful, interactive implementation of Conway's Game of Life built with React and HTML5 Canvas.  
  Features glowing neon cells, classic pattern presets, and full playback controls.  
  Active webpage: [willtech.github.io/game-of-life/#Gosper Glider Gun](https://willtech.github.io/game-of-life/#Gosper%20Glider%20Gun) 

- [**Text Transposer**](https://github.com/Willtech/transposer)  
  A client-side tool that remaps letters into a custom cipher.  
  Built with HTML, CSS, and JavaScript.  
  Active webpage: [willtech.github.io/transposer](https://willtech.github.io/transposer/) 

- [**BitPay**](https://github.com/Willtech/BitPay)  
  A project exploring blockchain payment monitoring and address mempool tracking.  
  Built with PHP, HTML, and JavaScript.  
  Active webpage: [bitpay.willtech.com.au](https://bitpay.willtech.com.au/)

- [**FWIDATA**](https://github.com/Willtech/FWIDATA)  
  A Bash utility that generates a companion script called **fwidata**, embedding a live snapshot of your system information (via neofetch) along with metadata headers, versioning, and helpful command‑line switches.
